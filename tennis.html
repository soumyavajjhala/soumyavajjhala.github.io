Since one of my professional skills is in Microsoft Office applications (Excel, Word, and PowerPoint), I thought it would be cool to apply a professional skill to my personal interest. Tennis has always been my favorite sport to watch and follow. There’s data involved with the number of tournaments played, number of titles won, wins, losses, and winning percentile ranges. So, I created an ongoing, long-term, YTD (year-to-date) portfolio, based on the sport.

For the number of the tournaments played, number of titles won, wins, losses, and winning percentile ranges, I knew that it would be best to organize this data is in Excel. I wanted to know where each women's singles Grand Slam Champion is ranked, based on her winning percentile ranges from each year. As a result, I collected data for the following women:

After I collected the data for each woman, I put everyone's data together and created a new separate Excel spreadsheet, named "Women's Singles Grand Slam Champions". This has each woman's number of tournaments played, number of titles won, wins, losses, and winning percentile range. All those pieces of data start from the year each woman made her debut on the WTA (Women's Tennis Association) tour until the current year.
